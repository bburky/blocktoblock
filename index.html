Description

An HTML5/JavaScript puzzle game of sliding blocks. Collect all blocks while trying to reach the goal. Dual controls of W,S,A,D and arrows are used to control two player-controlled blocks. Sliding blocks can be temporarily frozen by holding space. The game has a tutorial and guides the player into the puzzles.

Compatibility

Tested on current versions of Chrome.
